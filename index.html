🎬 Universal Stream Player — HLS / DASH / MP4 / Embed

Paste a URL above and hit Load.
Supports: native HLS (Safari/iOS), hls.js fallback for everything else (Chrome/Firefox/Edge), dash.js for MPEG-DASH (.mpd), direct MP4/WebM playback, and iframe embed fallback for embed pages (e.g. vidsrc-style players) that aren't raw media files. For CORS-blocked streams, enable "Use Proxy" and supply a working proxy prefix (default corsproxy.io). Some sites require a matching Referer header which browsers can't spoof client-side — for those, route through your own Cloudflare Worker/proxy.
