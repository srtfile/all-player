🎬 Universal Stream Player — HLS / DASH / MP4 / Embed

Paste a URL above and hit Load.
Supports: native HLS (Safari/iOS), hls.js fallback for everything else (Chrome/Firefox/Edge), dash.js for MPEG-DASH (.mpd), direct MP4/WebM playback, and iframe embed fallback for embed pages that aren't raw media files. For CORS-blocked or referer-locked streams, enable "Use Proxy" — the proxy worker expects ?url=BASE64(url)&headers=BASE64(JSON headers), and Referer/Origin entered above are packed into that headers payload automatically. All HLS sub-requests (manifests, key files, segments) are also routed through the proxy via hls.js's loader hooks, not just the initial URL, so multi-level / encrypted / relative-path playlists keep working without re-typing the proxy per segment.
